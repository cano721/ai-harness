# Planning Team Assets

`teams/planning` contains both legacy drafts and the new planner-mode bundle.

- `skills/` and `CLAUDE.md` are older planning assets kept for reference while they are still being reviewed.
- `bundle/` is the installable source used by planner mode.

When you update planner-mode installation, change files under `bundle/`.
Do not treat `teams/planning/skills` as the shipping source for planner mode.
