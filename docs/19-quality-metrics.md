# AI Harness - 품질 & 채택 메트릭

## 메트릭 분류

### 1. 채택 메트릭 (Adoption)

| 메트릭 | 산출 방법 | 목표 |
|--------|----------|------|
| **설치율** | 하네스 설치 프로젝트 / 전체 프로젝트 | > 80% |
| **활성 사용률** | 최근 7일 하네스 경유 세션 / 전체 세션 | > 90% |
| **바이패스율** | 바이패스 세션 / 전체 세션 | < 5% |
| **팀별 채택** | 팀 프로필 사용 비율 | 전 팀 100% |
| **버전 최신율** | 최신 버전 사용 프로젝트 / 전체 | > 70% |

### 2. 안전 메트릭 (Safety)

| 메트릭 | 산출 방법 | 목표 |
|--------|----------|------|
| **차단 성공률** | 실제 위험 차단 / 위험 시도 | 100% |
| **오탐률 (False Positive)** | 정상 요청 차단 / 전체 차단 | < 5% |
| **미탐률 (False Negative)** | 위험이 통과 / 위험 시도 | 0% |
| **보안 사고 건수** | 하네스 우회로 인한 사고 | 0건 |
| **시크릿 유출 차단** | secret-scanner가 차단한 건수 | 추적 |

### 3. 효율 메트릭 (Efficiency)

| 메트릭 | 산출 방법 | 목표 |
|--------|----------|------|
| **Hook 지연 시간** | 전체 Hook 체인 실행 시간 | < 500ms (p95) |
| **에이전트 처리량** | 하네스 경유 시 vs 미경유 시 작업 완료 시간 | 지연 < 10% |
| **자동 검증 통과율** | 첫 시도에 lint/test/보안 통과 비율 | > 85% |
| **재작업률** | Hook 차단 후 수정 재시도 비율 | 추적 |

### 4. 품질 메트릭 (Quality)

| 메트릭 | 산출 방법 | 목표 |
|--------|----------|------|
| **컨벤션 준수율** | 규칙 위반 없이 완료된 작업 비율 | > 90% |
| **PR 리뷰 코멘트 감소** | 하네스 도입 전후 리뷰 코멘트 수 비교 | -30% |
| **AI 생성 코드 수정율** | 리뷰에서 수정 요청받은 비율 | < 20% |

### CLI로 조회

```bash
# 전체 요약
$ ai-harness metrics

# 특정 카테고리
$ ai-harness metrics --category safety
$ ai-harness metrics --category adoption

# 기간 지정
$ ai-harness metrics --from 2026-03-01 --to 2026-03-18

# 팀별
$ ai-harness metrics --by team

# 내보내기
$ ai-harness metrics export --format csv > metrics-march.csv
```

## 데이터 수집 방법

```
감사 로그 (JSONL)
    ↓ 집계
메트릭 저장소 (.ai-harness/metrics/)
    ↓ 시각화
대시보드 (CLI / 웹)
```

### 집계 스케줄

| 주기 | 집계 항목 |
|------|----------|
| **실시간** | 차단 건수, 비용, 에러 |
| **시간별** | Hook 지연 p50/p95/p99 |
| **일별** | 채택률, 오탐률, 팀별 비교 |
| **주별** | 주간 리포트 자동 생성 |
| **월별** | 트렌드 분석, 목표 대비 달성률 |

## 알림 & 이상 감지

| 조건 | 알림 |
|------|------|
| 오탐률 > 10% | Team Champion + Owner에게 알림 |
| Hook 지연 p95 > 2초 | Owner에게 알림 |
| 바이패스율 > 10% | Owner + 보안팀 알림 |
| 특정 팀 채택률 < 50% | 해당 Team Champion 알림 |
| 주간 비용 전주 대비 +50% | 해당 팀 알림 |
